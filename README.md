## Descripción

TechStore es un proyecto educativo Full Stack desarrollado con el objetivo de consolidar conocimientos en desarrollo web moderno utilizando React, Node.js y MySQL.

El proyecto consiste en una pequeña tienda online donde los usuarios podrán visualizar productos, registrarse, iniciar sesión y realizar compras mediante un carrito de compras.

---

## Objetivos del proyecto

- Reforzar fundamentos de JavaScript.
- Comprender la arquitectura de una aplicación Full Stack.
- Aplicar React para el desarrollo del Frontend.
- Construir una API REST con Node.js y Express.
- Integrar una base de datos MySQL.
- Implementar autenticación de usuarios.
- Desarrollar un CRUD completo de productos.
- Aplicar buenas prácticas de organización y reutilización de componentes.
- Publicar el proyecto en GitHub como parte del portfolio profesional.
