# 🛒 TechStore

TechStore es una aplicación desarrollada con React y Vite que simula una tienda de productos tecnológicos.

## 🚀 Tecnologías utilizadas

- React
- JavaScript
- Vite
- CSS
- Git
- GitHub

## 📋 Funcionalidades

- Catálogo dinámico de productos.
- Renderizado mediante map().
- Componentes reutilizables.
- Uso de Props.
- Disponibilidad según stock.
- Imágenes de productos.
- Diseño con tarjetas (Cards).
- Navegación básica.

## 📦 Productos

- TV
- Radio
- Notebook
- Mouse
- Teclado
- Monitor
- Celular
- Reloj
- Zapatillas
- Tablet
- Control
- Proyector

## 🧠 Conceptos de React aplicados

- Componentes
- Props
- Renderizado condicional
- Arrays de objetos
- map()
- Organización de carpetas
- Manejo de assets (imágenes)
